# Security Policy

## Reporting a vulnerability

If you discover a security issue in Skipper, please **do not open a public issue**.

Instead, email **jerryeagbesi@gmail.com** with:

- A description of the issue
- Steps to reproduce
- The version of Skipper affected

You can expect an initial response within a few days. Once the issue is confirmed, a fix will be prepared and released, and you'll be credited in the release notes (unless you prefer to remain anonymous).

## Supported versions

Only the latest release receives security updates.
